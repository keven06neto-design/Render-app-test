# 🪙 Simulador de Lançamento de Moeda

## 📋 Descrição
Aplicativo web interativo desenvolvido com Streamlit que simula o lançamento de uma moeda N vezes, demonstrando a Lei dos Grandes Números.

## 🛠️ Tecnologias Utilizadas
- Python
- Streamlit
- Pandas
- SciPy
- Git & GitHub
- Render (deploy)
